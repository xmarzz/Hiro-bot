# Hiro Bot

![Hiro Bot](https://github.com/xmarzz/Hiro-bot/assets/90947733/3647343b-d755-435d-a408-5279eace31d6)

Hiro Bot is designed to enhance your productivity by helping you manage your daily tasks efficiently.

## Current Features

### Flow Time

**Description:** Flow Time is a method that involves scheduling specific time blocks for different tasks or types of work throughout the day. Each block is dedicated to a particular activity, and you switch between tasks at the end of each block.

**How It Works:**
1. Identify your most important tasks for the day.
2. Allocate specific time blocks to those tasks.
3. Set a timer for each block, typically ranging from  25 to  60 minutes, with  45 minutes being the default.
4. Focus solely on the task during that block, and take a short break between blocks.

**Advantages:**
- Provides flexibility in adjusting the length of time blocks based on task complexity.
- Allows for deep focus on one task at a time.
- Can accommodate various types of tasks and energy levels.

**Example Commands:**
- `/new_session <name>` - to start a new session
- `/check_session` - to check the current session
- `/end_session` - to end the current session
- `/about` - to learn more about the bot

## Getting Started

1. Clone this repository and run `npm install`.
2. To run the bot, execute `node index.js` or `npm start`.
